== Description ==

This plugin is for redirect all 404 pages to home page or any custom page, 404 URLs which can negatively effect search engine indexing. this plugin to solve these errors in SEO way by redirecting them using SEO 301 redirection.

**How to use the plugin?**

<p>Our Plugin will try to redirect all 404 pages to home page or any custompage base on 301 rediction </p>
<p>After installing the plugin, enabled status from control panel above, if you want to disable the plugin functions, just set the status to disabled.</p>

== Installation ==

Extract the zip file and just drop the contents in the wp-content/plugins/ directory of your WordPress installation and then activate the Plugin from Plugins page.

After active plugin go to Setting->All 404 Redirect to Homepage and enable status for redirection.
